    it 'should convert text nodes to strings', ( done ) =>

HTML text nodes should serialize as ordinary strings. We test a variety of ways they might occur.

            textNode = document.createTextNode 'foo'

                textNode.toJSON()

                div.childNodes[1].toJSON()
